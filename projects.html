Software Engineering Projects:

This is an about section for about my projects

Piano React Visualizer Project: Piano Link This is a piano visualizer web application built on react. Whenever you click a note or press down with a key it will play the note and create a note animation that illustrates which note was played. Additionally you can record yourself playing and download the file as a mp4. There is also playback capabilities for a midi file

Apartment Web Scraping Visualizer: This project web scrapes apartment data (price, # of beds, location) using Python (Beutiful Soup, Selenium) and then does prelminary cleaning with Pandas. The data is then downlaoded into csv in incremental chunks and further joined and cleaned in SQL. Then it is loaded into a django backend and displayed using chart js
