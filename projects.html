Volleyball Database SQL Project This project demontrates my skill with SQL DDL as well as SQL queries by constructing a database managemnent system with 20+ 3NF normalized tables representing elements such as Teams, Players, Coaches etc. After over 100 tuples were inserted a wide range of SQL queries can be executed such as updates, select, group by aggreate, division queries and more.

Piano Visualizer This is a piano visualizer web application built using React. It utilizes the component-based architecture of React to effectively build a working piano.

Apartment Web Scraper Dashboard This project web scrapes apartment data (price, # of beds, location) using Python (Beautiful Soup) for the 25 most populous cities in the US. The data is then cleaned and wrangled using Pandas into a single csv file. Finally, a dashboard was created in Tableau to show a heat map of the median apartment rent as well as a horizontal bar graph comparing the median rent to the median income for those cities.

Django Country Search This is a Django project of a Searchable Country Database. The Country database is a MySQL table supporting utf8mb4 character set and rendered using an HTML table. Through this project I developed familiarity with Django structure as a backend framework and building HTML templates, Model classes, and routing URLs.
